## DESCRIPTION:

This project is a fundamental part of the second Milestone from the 42 School, that cannot be skipped. During its execution, you will learn how to correctly use linked lists in C, parse and sort data, and create different kinds of algorithms. In the specific case of the chosen algorithm for the development of this program (radix), you will also learn about binary, and bitwise operators. 

## INSTRUCTIONS:


The project compiles using a MAKEFILE in which the necessary compilation rules for C programs are declared, as well as the corresponding flags.

So, in order to compile the project, you will only need to use the 'make' command.

To execute the program, you will have to use the following command:

`./push_swap`
